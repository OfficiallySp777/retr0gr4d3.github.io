The requirements to get here...

  • Hardware
  • Software
  • Time

The fun part. For this, most people fire up a gaming PC or use Google Cloud. Obviously, beefier the specs, faster the build. I'd say your minimum requirements are 8GB RAM, 500GB Storage and a processor that doesn't cry when you try to run more than 1 tab of Google Chrome. I have 500GB NVMe, 8GB RAM, i7-8550u clocked at 4GHz. This equals roughly 1h build times.

Linux. Whether this is host (recommended) or virtualised, Linux is necessary to build Android. I recommend the latest LTS of Ubuntu if you're not too familiar.

Time is an infinite source. Its up to you how you spend it. I recommend familiarising yourself with this documentation before trying to go straight at it - that's the mistake I made. I didn't know anything, jumped straight in and wasted a LOT of time.

Getting started.
Installing the tools needed for the job.

To achieve this, you are going to need to install multiple tools. Assuming you have already installed Linux, fire up the Terminal (CTRL + T / Terminal Icon) and run the below, as sudo.

apt install bc bison build-essential repo git ccache curl flex g++-multilib gcc-multilib git gnupg gperf imagemagick lib32ncurses5-dev lib32readline-dev lib32z1-dev liblz4-tool libncurses5 libncurses5-dev libsdl1.2-dev libssl-dev libwxgtk3.0-dev libxml2 libxml2-utils lzop pngcrush rsync schedtool squashfs-tools xsltproc zip zlib1g-dev openjdk-8-jdk
